# Tower-Defence-Game

## Game Name
TowerDefenceWarriors

## Description
This is a school project for the class **Programming Language Principles**. The game is a tower defense game where players defend their base from waves of enemies by strategically placing towers.

## Features
- **Godot Game Integration**: The game is embedded in a web application.
- **Leaderboard**: Displays the top 10 scores.
- **Towers and Enemies**: Various towers and enemies with unique abilities.

## Prerequisites
- **Docker**: Ensure Docker is installed on your system.
- **Node.js**: The backend is built using Node.js.
- **SQLite**: The database used for storing game data.
